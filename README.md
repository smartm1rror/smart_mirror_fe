### Server Start
```
yarn dev
```

### 개발 요소 
#### 시스템 구성
- [ ] 얼굴 인식
    - [ ] 카메라 켜기
    - [ ] 사용자 얼굴 인식
        - [ ] 분석을 원하시면 3초간 가만히 있어주세요!
- [ ] 데이터 전송
    - [ ] 얼굴 사진 데이터 백엔드 전송
    - [ ] 전송 후 카메라 일시 중지
- [ ] 데이터 수신
    - [ ] 수신된 데이터 가공
    - [ ] 데이터 사용자에게 출력

    - [ ] 시간 (우측 상단)
    - [ ] 날씨 (좌측 상단)
- [ ] 디자인
    - [ ] 얼굴 인식 과정에서 사용자 어그로
    - [ ] 데이터 가공 중 일시적으로 대기 요청
